    • Description

    SoundCloud Enhancer offers loads of customizations enhancing the SoundCloud experience

    Features

  • Dark mode
  • Fullwidth mode
  • Responsive design
  • Toggle sidebar
  • Toggle sqaure profile pictures
  • Enhanced "more actions" menu
  • Display mode (Default, compact and grid)
  • Mass unfollower
  • Hide playlists from stream
  • Hide previews from stream
  • Hide longer tracks form stream
  • Hide specific tags from stream
  • Hide all reposts from stream
  • Hide your activity from stream
  • Hide other clutter
  • And more to come...
